# ARFollowObject

ARKit による仮想オブジェクトがカメラに追従する UI。  
仮想オブジェクトをタップするとカメラに追従し、再度仮想オブジェクトタップすると追従が解除される

[Blog](http://appleengine.hatenablog.com/entry/2018/12/21/153437)
